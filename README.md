Syncmotic es una plataforma web de domótica pensada para negocios como cafeterías, bares y pequeños comercios que quieren ofrecer espacios más inteligentes y eficientes sin complicarse con la tecnología.  

El objetivo del proyecto es centralizar, en una sola interfaz sencilla, el control y la supervisión de distintos sistemas del local: iluminación, red Wi-Fi, ambiente (música, clima), energía y dispositivos conectados. De esta forma, el administrador puede ajustar la experiencia del cliente, ahorrar consumo eléctrico y mantener todo el entorno bajo control en tiempo real.

Actualmente, Syncmotic se enfoca en:

- Proveer un panel limpio y minimalista para visualizar el estado general del local.
- Integrar de forma gradual distintos módulos de domótica (iluminación, red, sensores, etc.).
- Ofrecer una base preparada para escalar a múltiples locales o franquicias.
- Ser fácil de desplegar en entornos modernos (Docker) para su uso en producción.

El proyecto está construido con **Vite + React + TypeScript** y una interfaz minimalista inspirada en aplicaciones modernas SaaS, pensado para ser el núcleo de futuras integraciones de hardware y servicios de automatización.
